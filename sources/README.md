## Instalation

- Install openGL:
```bash
apt install freeglut3-dev
```

- Install JPEG :
```bash
apt install libjpeg-dev
```

- Install LXmu :
```bash
apt install libxmu-dev libxmu-headers
```

- Install LXi :
```bash
apt install libxi-dev
```

- Install Logger :
```bash
apt install libspdlog-dev
```
